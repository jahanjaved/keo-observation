KEO Safety Awareness Website - Email Play Button Version

Most email clients do not play MP4 directly inside the email.
Use a play-button image in the email and hyperlink it to the direct-play page.

Direct play pages:
index.html?video=english&play=1
index.html?video=arabic&play=1
index.html?video=hindi&play=1

If browser blocks autoplay, the user will still see a big play button immediately.
Upload all files and the assets folder to GitHub Pages root.
